安装git

# Git安装配置

# 下载git

git官网

git-windiowsx64-1.30.1

  • Linux与Mac安装

linux和Mac都是直接使用命令行安装。 下面是Debian安装步骤

$ sudo apt update
$ sudo apt upgrade
$ sudo apt install git -y
1
2
3

Mac可以使用 homebrew安装

$ brew install git
1

提示

Mac在安装xcode时会自动安装git,使用homebrew需要先安装，不了解的可以自行百度一下.

  • Windows安装

Windows 安装git可以一路next就行了，安装最后别忘记勾选把git加入系统变量.

  • Git 配置

git可以使用http和ssh协议连接，看个人习惯，本人喜欢使用ssh。

  1. 生成密钥

Windows下，安装完git客户端后，在桌面任意位置右键，点击git bash here.

进入bash后，输入命令ssh-keygen.exe -t rsa -b 4096 -C ‘your email’win10环境）

# windows
$ ssh-keygen.exe -t rsa -b 4096 -C xxx@qq.com

# mac or linux
$ ssh-keygen.exe -t rsa -b 4096 -C 
'your email'
1
2
3
4
5
6

提示

在Mac或者Linux下，你可能需要sudo权限.

输入命令后一直回车就行了，最后在个人文件夹下应该会有三个文件，id_rsa,id_rsa.pub,known_hosts,Windows一般是在C/User(用户)/.ssh/文件夹下 Linux或Mac可以直接在命令行找到

# 查看共钥的内容复制
$ cat id_rsa.pub
1
2
  1. 关联本地git和github

打开github-> 登录 -> 点击右上角头像 -> Settings -> SSH and GPG keys

  1. 测试连接
$ ssh -T git@github.com
1

如果不出意外的话，你现在就可以愉快的使用git和github了.
